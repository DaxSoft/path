# routes()


**returns** `array`

> It returns all routes from the instance

<hr>

## Example

``` js

// Take at account the example from 'set.md'

instance.routes()

// Output

[
    {
        name: 'src',
        path: __dirname
    },

    {
        name: 'example',
        path: '../example'
    }
]

```
